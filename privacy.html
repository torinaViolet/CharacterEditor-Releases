Privacy

隐私说明

最后更新：2026 年 7 月 14 日

本地数据

Character Editor 将角色、世界书、预设、设置与聊天会话保存在你的设备上。公开发行仓库不接收这些内容。

外部服务

当你主动配置 AI 供应商、远程 MCP Server 或其他外部接口时，请求内容会发送到你选择的服务。相关服务如何处理数据取决于其各自的隐私政策。

本地文件能力

内置文件 MCP 只能访问你明确授权的目录。写入、移动和删除等操作默认需要确认。

更新检查

应用可能访问本公开仓库以检查新版本和下载安装包。该过程由 GitHub 提供网络服务。

建议

请勿将敏感 API Key、私人角色数据或会话数据库上传到公开仓库。使用第三方服务前，请确认其数据政策符合你的需求。
